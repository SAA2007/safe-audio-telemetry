# Ethical Use Policy

## Purpose
This project processes audio data, which is inherently sensitive.  
This document defines the **ethical boundaries and intended use** of the software.

## Consent & Transparency
- Audio recording must occur **only with informed and explicit consent**
- The software must provide a **visible foreground notification** when active
- Users must have the ability to **start and stop recording intentionally**
- Hidden, background, or deceptive operation is not supported

## Permitted Use
This software may be used only:
- On devices owned by the user, or
- With the clear, informed consent of all recorded participants
- For educational, research, or personal experimentation purposes

## Prohibited Use
The following uses are **explicitly disallowed**:
- Recording individuals without their knowledge or consent
- Surveillance, tracking, or monitoring of others
- Use in coercive, abusive, or exploitative contexts
- Commercial deployment or data collection **without explicit approval**
- Any use intended to evade laws, platform safeguards, or user awareness

## Responsibility
Users are solely responsible for ensuring lawful and ethical use of this software.  
The developer does not endorse, support, or condone misuse of the project.
